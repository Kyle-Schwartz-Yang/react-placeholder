# Modals Provider

> Якщо ми захочемо відкривати модальне вікно у модальному вікні, тут є невеликі проблеми : на першому модальному вікні ми блокуємо скролл, на другому він розблоковується, а потім зовсім ламається, звісно так не повинно бути, але поки що як є.

#### `feature/modals-provider`

- Модальне вікно, зроблено через Context API (Modal Provider)
- Є анімація появи за допомогою framer motion
- Модальне вікно відкривається за допомогою назви що не дуже єффективно, краще відкреття за змістом
